All Pills é uma aplicação concebida para tornar a gestão de medicação mais simples, eficaz e eficiente.
Esta permite que os utilizadores registem/programem facilmente os horários e doses dos seus medicamentos diretamente no seu dispositivo móvel.
A informação é, então, sincronizada com uma caixa de medicamentos inteligente, equipada com um Arduino incorporado.
Quando chega a hora de tomar um comprimido, a caixa emite um alerta sonoro e visual para lembrar o utilizador de que é a hora de administrar o medicamento.
